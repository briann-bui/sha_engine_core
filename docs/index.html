SHA-256 Engine IP Core

Pure-digital SystemVerilog hash engine with synthesizable RTL, direct port-level interface, and AES-aligned project structure.

SystemVerilog SHA-256 UVM Synopsys VCS

Overview

Algorithm
SHA-256, with SHA-224 initial values reserved as a placeholder.
Input
512-bit pre-padded message block in big-endian word order.
Output
256-bit digest as {H0,H1,H2,H3,H4,H5,H6,H7}.
Verification
UVM direct-port smoke environment aligned to the AES repository layout.

Architecture

Interface

sha2_256_core
├── control: start, init, next, final, mode
├── input:   block_valid, block[511:0], msg_bit_len[63:0]
└── output:  block_ready, busy, done, error, digest_valid, digest[255:0]
